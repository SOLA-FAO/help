لوحة المراقبة والقائمة الرئيسية

سوف يتم فتح شاشة لوحة المراقبة مباشرة بعد تسجيل الدخول . تعرض لوخة المراقبة الطلبات المعينة للمستخدم الخاص بك كما تعرض الطلبات التي لم يتم تعيينها بعد



باستخدام القائمة الرئيسية وشريط الادوات تستطيع تنفيذ المهام التالية;

  • الخروج من النظام (خروج او اغلاق النافذة الاساسية)
  • تغيير اللغة (عرض - اللغة)
  • اختيار مستوى التدقيق المطلوب
  • العودة الى لوحة المراقبة (اوحة المراقبة > لوحة المراقبة او )
  • انشاء او ايداع طلب جديد (الطلبات > طلب جديد )
  • البحث عن طلب موجود ( البحث > البحث عن طلب او Application search)
  • البحث عن ملكية جديدة ( البحث > بحث عن ملكية او Property search)
  • البحث عن وثيقة موجودة ( البحث > البحث عن وثيقة او Document search)
  • البحث عن شخص موجود او اضافة شخص ( Party)
  • فتح متصفح خرائط سولا ( الخارطة > Ctrl + M او )
  • Vعرض شاشة حول النظام ( مساعدة > حول …)
  • افتح ملف المستخدم لتغيير كلمة المرور ( User profile)
  • افتح مساعد سطح المكتب (مساعدة > المساعدة بالسياق )
  • عرض تقرير الايداع ( التقارير >تقرير الايداع )
  • تحضير قوام العرض العامة (التسجيل المنتظم > اخطارات العامة )
  • Pتحضير مخططات العرض العام (التسجيل المنتظم – اخطارات العامة )
  • شهادات التسجيل المنتظم (التسجيل المنتظم – الشهادات )
  • تقارير التسجيل المنتظم (التسجيل المنتظم – التقارير)
  • اختيار وتعيين طلب غير معين ( قائمة النقر على زر الفأرة الأيمن او )
  • اختيار وتعديل طلب معين او غير معين ( قائمة النقر على زر الفأرة الأيمن او)
  • احتيار والغاء تعيين طلب ( قائمة النقر على زر الفأرة الأيمن او )
  • تحديث قائمة الطلبات ( قائمة النقر على زر الفأرة الأيمن او )

انظر ايضا

  • تعديل بيانات طلب
