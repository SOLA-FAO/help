Property Search

The Property Search screen allows you to find and view properties of interest. It can be accessed from the Search >  Property menu or selectingProperty search

Property Search

You can search by any combination of Name first part, Name last part or Right holder (name). These fields also support partial and full matching. To view the details for a property, select the property in the Search results list and clickOpen. To clear the search criteria from a search tab, click Clear.
