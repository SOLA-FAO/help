Property Tasks

Property tasks can be used to create, maintain or dispose state land property records.

Task

Description

Record New Property

Used to create a new state land property and record any type of interest for the new property as well as create unmapped state land parcels (using a metes and bounds description) and property relationships.  

Maintain Property

Used to edit the details of an existing state land property including any interests, parcels or property relationships.

Manage Valuations

Used to capture new valuations for one or more existing state land properties. 

Dispose Property

Updates the status of a state land property to indicate it has been disposed.
