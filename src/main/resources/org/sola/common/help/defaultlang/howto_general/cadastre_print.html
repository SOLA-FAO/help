Create a Cadastre Print

The Cadastre Print service is used to record a request from an agent or member of the public for a map printout. The steps for creating a cadastre print are similar to those for performing a Title Search with the following exceptions

    • When creating the application add a Cadastre Print service instead of the Title Search service
    • To produce the map printout, use the Print tool on the SOLA Map Viewer.
