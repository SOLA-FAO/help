Vary Right or Restriction

If you need to change the details of a registered right, use the Vary Right (General) service. This service allows you to change the details for any right or restriction.

Steps

    1.  Start the Vary Right (General) service. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the right you intend to change. If the right is current, theVary too will enable. Click  Vary to open the right screen. 
    3. Update the right details as appropriate and set the Notation text. If the right was migrated from another system, you may need to populate some of the right details before you can save your changes.
    4. ClickCreate & Save and close the right screen.
    5. Varying a right creates a new pending version of that right. If you need to make further changes to the right information, select the pending right in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending right details completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
