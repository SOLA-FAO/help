Map Tools

The tools provided for the SOLA Map Viewer are

Zoom to extent

Zoom out

Zoom in

Pan

Information tool

 Measure

Print 

Export to KML

Access to the Measure, Print and Export to KML tools is restricted and they will only display if you have the appropriate security privilege. 

The SOLA Map Viewer can also support additional map tools for editing and related spatial tasks. These tools are described in the relevant screen descriptions.
