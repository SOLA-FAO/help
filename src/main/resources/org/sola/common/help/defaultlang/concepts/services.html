Tasks

Tasks are a key component of SOLA State Land as they control the changes that can be made to the properties and parcels maintained by SOLA State Land as well as supporting common activities the state must perform when dealing with state land transactions such as public display of relevant information, formal notifications and property valuations. Every job lodged in SOLA State Land must include at least one task. In cases where there several changes required to property or parcels as part of a single job, you can choose to add multiple tasks to a job. The categories of tasks supported by SOLA State Land are

    • Parcels
    • Property
    • Leases and Licenses
    • Interests
    • Claims
    • Job
