Attacher un Document Numérique

Pour attacher un document numérique, utiliser l'onglet Document dans l'écran de détail de la demande pour ajouter un nouveau document à la demande ou sélectionner un document existant de la liste et cliquer sur le bouton Attacher.



Le navigateur de fichiers s'affiche pour permettre de sélectionner un document numérique du système de fichiers local ou d'une localisation externe configurée pour SOLA.





Le disque à distance affiche les documents qui ont été placés dans un système de fichiers en réseau spécifique (ex: un dossier configuré pour un scanner rattaché en réseau). for a network attached scanner. La prévisualisation des vignettes sont possibles pour les images de format standard (png, tiff, jpg) ainsi que les PDF. Une fois le document à attacher choisi, cliquer sur le bouton Attacher. Une icône trombone s'affiche dans la liste des documents de l'onglet Document pour indiquer les documents ayant été attachés. Enfin, il est nécessaire de sauvegarder la demande pour que les attachements restent liés à la demande.

Noter que les fichiers de moyenne et grosse taille peuvent créer des problèmes dans le traitement de l'attachement. Pour plus de détails, se référer aux Problèmes Connus.
