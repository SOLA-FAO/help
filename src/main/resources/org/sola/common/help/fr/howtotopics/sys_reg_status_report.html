Prepare Status & Work Reports

Two types of reports can be generated. One that reflects the status of all systematic registration applications, block by block at the time of the report. The second that records the changes that have occurred to services related to systematic registration in a particular time interval.
Both reports cover all applications with a systematic registration service and are organised block by block.

Steps

  1. From the Dashboard form click on the Systematic Registration â€“ Public Display menu Report. From a further sub menu option select the type of report you want to generate; Status or Work.
  2. When a Status report is selected the report will generated and displayed as a pdf document that can be printed or saved.
  3. When a Work report is selected a screen will be displayed for the dates defining the period to be reported on. Once the dates have been entered. Click on the View Report button and the report will be displayed as a pdf document that can be printed or saved
