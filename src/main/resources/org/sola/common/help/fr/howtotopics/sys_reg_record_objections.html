Systematic Registration Record Objections

To record objection (including counter claims) details you will need to access the application recording the systematic registration details for that property. Select the application in the assigned or unassigned application lists on the Dashboard and choose Edit. If the application is not displayed in the application lists, use the Search Application screen to locate it and click the Edit button.
This will open the Application Details screen allowing you to edit the application details and save changes

Steps

  1. Open the application and assign the application with the Systematic Registration Claim service to yourself.
  2. Open the assigned application and on the Services tab click on + and double click on “Objection” service
  3. On the Documents tab add the Objection form and any other supporting documents associated with the disputed rights. Ensure all recorded documents have a linked scanned image
  4. Start the Objection service by selecting it and clicking on
  5. On the Rights / Restrictions tab add a caveat restriction and identify the person making the dispute in the Rightholder form and also make sure their contact details are also recorded. You will also need to Add a document to the caveat restriction and add the Objection Form document. .
  6. Save changes to Property Details and Complete the Systematic Registration Dispute service.
