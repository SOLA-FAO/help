Известные проблемы

Известные проблемы для релиза SOLA Alpha следующие:

  1. Прикрепление больших файлов (5Мб и более) может привести к зависанию SOLA, в результате чего может потребоваться перезапуск приложения
  2. При попытке открыть прикрепленный файл среднего размера (2-5Мб) может привести к зависанию клиентского приложения SOLA, до тех пор пока документ полностью не загрузится. Это может занять 10-90 секунд.
  3. Статус услуг не обновляются на "исполняется", после запуска услуги на вкладке "Услуги". Это не мешает продолжить работу с услугой.
  4. Удаление / прекращение права (например, право собственности) не переводит право в историю.
