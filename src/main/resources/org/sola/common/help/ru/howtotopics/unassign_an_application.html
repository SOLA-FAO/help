Освободить от исполнения заявления

Для того чтобы снять заявление(я) с исполнения, отметьте его галочкой в списке назначенных заявлений и нажмите на кнопку "Освободить" (или щелкните правой кнопкой мышки и выберите пункт из контекстного меню). В появившемся предупреждении о снятии с исполнения, нажмите кнопку "ОК". В результате этого, заявление будет снято с исполнения и перемещено в список неназначенных заявлений.
