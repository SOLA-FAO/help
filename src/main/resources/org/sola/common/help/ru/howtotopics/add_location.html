Добавление расположения для заявления

Вы можете указать приблизительное местоположение на карте, где будут выполняться услуги по заявлению. Для этого необходимо перейти на вкладку "Точки карты" в форме заявления и воспользоваться инструментом "Добавить точки" . Данный инструмент позволяет добавить более одной точки расположения заявления в случае если услуги по заявлению затрагивают большую площадь или распределены в разных частях карты. Точки расположения представлены в виде зеленых кружков.



Точки расположения могут быть полезны для определения других заявлений находящихся поблизости и возможно имеющих отношение или влияние на текущее заявление.

Инструмент удаления точек расположения может быть использован для удаления точек расположения добавленных для заявления.
