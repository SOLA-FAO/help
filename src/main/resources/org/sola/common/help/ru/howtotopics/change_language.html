Смена языка

Приложение SOLA Desktop поддерживает различные языки для отображения надписей, сообщений, форматов дат и чисел, а также выпадающих списков на пользовательских формах (справочники). Для смены языка в главном окне выберите меню Вид => Язык и в появившемся списке выберите нужный язык. Вам понадобится перезапустить приложение, о чем будет сообщено в появившемся предупреждении.
