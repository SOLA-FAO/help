Регистрация нового заявления

Для создания/регистрации нового заявления, выберите главное меню Заявления > Новое заявление или нажмите на кнопку в панели инструментов Новое заявление.

Для нового заявления необходимо предоставить как минимум контактные данные заявителя (имя, фамилия, пол, адрес) и добавить одну услугу. Остальная информация может быть заполнена по необходимости. После ввода всех данных, сохраните заявление. В результате этого будет сгенерирован номер заявления и отображено уведомление о подачи заявления, которое можно распечатать и предоставить клиенту.
