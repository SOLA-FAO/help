Расчет стоимости

Для расчета стоимости по заявлению, перейдите на вкладку "Оплата" и нажмите кнопку "Вычислить стоимость" на панели инструментов формы заявления.


Расчет стоимости не приводит к сохранению изменений в заявлении. Данная функция позволяет определить стоимость услуг, перед тем как заявление будет зарегистрировано в системе. Расчет стоимость также производится автоматически при первом сохранении заявления.
