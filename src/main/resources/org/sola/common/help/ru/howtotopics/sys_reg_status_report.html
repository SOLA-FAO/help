Подготовка отчетов о статусе и проделанной работе

При помощи данной функции можно подготовить два типа отчетов. Один для отображения статуса всех заявлений на системную регистрацию на дату отчета, сгруппированных поблочно. Второй для отображения услуг системной регистрации по которым были сделаны изменения в системе в указанный период времени.

Шаги

  1. В главном окне SOLA Desktop выберите меню "Системная Регистрация" => "Отчеты". Из последующего подменю выберите тип отчета, которые вы хотите получить (Статус или Выполнение).
  2. Для получения отчета по статусу заявлений, введите блок системной регистрации и нажмите кнопку "Просмотр Отчета".

  3. Для получения отчета по исполнению услуг, введите временной интервал и блок системной регистрации. Нажмите кнопку "Просмотр Отчета" для генерации отчета.
