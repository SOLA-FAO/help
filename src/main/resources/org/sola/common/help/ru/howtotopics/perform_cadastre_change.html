Выполнение изменения кадастра

Вы можете выполнить изменение кадастра путем добавления услуги "Изменение кадастра" в заявление. Когда услуга будет запущена, появится форма изменения кадастра. Выберите необходимый участок, введите точки съемки используя инструмент "Создать / Изменить точки съемки" или введите координаты непосредственно на форме точек съемки. Как только все точки съемки созданы, используйте инструмент "Создать / Изменить точки съемки" для привязки граничных точек к точкам участка. После того как привязка закончена используйте инструмент "Новый участок" для создания нового участка по созданным точкам и сохраните изменения. Для завершения кадастровых изменений нажмите кнопку "Завершить" для услуги "Изменение кадастра" на форме заявления и затем одобрите само заявление. В результате этого, временные участки станут действительными (активными), а целевые участки получат статус "история" и не будут являться активными.
