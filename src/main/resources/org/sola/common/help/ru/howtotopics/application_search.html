Поиск заявления

Форма поиска заявления позволяет найти интересующие вас заявления для дальнейшего просмотра или редактирования. Данная форма может быть открытка через главное меню "Заявления" => "Поиск заявлений" или соответствующую кнопку на панели инструментов Поиск заявлений.

Вы можете осуществлять поиск используя комбинацию номера заявления, имя агента, имя контактного лица, период подачи заявления, номер документа прикрепленного к заявлению или его официальный номер зарегистрированный в SOLA. Номер заявления, имя агента, контактное лицо и номер документа поддерживают частичное и полное совпадение поиск. В таблице результатов поиска вы можете щелкнуть на заголовок одного из столбцов для сортировки строк по его содержимому. Повторный щелчок применить обратную сортировку.
Вы также можете открыть заявление из результатов поиска для дальнейшего редактирования или просмотра. Для этого выберите заявление и нажмите кнопку "Открыть", расположенную в панели инструментов над результатами поиска.
