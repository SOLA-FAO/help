Подготовка участков для публичного показа

На этом этапе подготавливаются карты для публичного показа. Участки имеют подпись с номером участка. Карта будет увеличена до границ области с участками, содержащихся в указанном блоке системной регистрации. Участки в прилегающих блоках будут отображены без номера участка.

Шаги

  1. В главном окне SOLA Desktop выберите меню "Системная Регистрация" => "Публичный показ карт". В появившемся окне просмотра карты, перейдите на вкладку "Карта Публичного Показа". Введите имя последней части участков, описание площади и срок общественного уведомления. Нажмите кнопку "Отцентровать Карту"
  2. Карта увеличится до границ всех участков, имеющих указанную последнюю часть кода участков.
  3. Выберите тип макета, введите текст надписи, которая будет отображаться в центре карты, введите масштаб и нажмите кнопку "Распечатать участок".
  4. Будет сгенерирован отчет, содержащий необходимую информацию. Вы можете распечатать его или сохранить в формат PDF для дальнейшего использования.
